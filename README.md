# 👨‍💼 HR Employee Dataset Analysis

📌 Overview

This repository contains a comprehensive **HR Employee Dataset** with over **2 Million employee records**. The dataset is designed for practicing **Data Analysis, HR Analytics, Business Intelligence, Data Visualization, and Machine Learning** techniques.

It provides valuable insights into employee demographics, job roles, performance ratings, salaries, work modes, and employment status.

---

## 🎯 Objectives

✅ Analyze workforce distribution

✅ Understand employee performance trends

✅ Study salary patterns across departments

✅ Explore remote, hybrid, and on-site work preferences

✅ Build predictive Machine Learning models

✅ Create interactive dashboards and visualizations

---

## 📊 Dataset Information

| Feature | Description |
|----------|-------------|
| Employee ID | Unique employee identifier |
| Full Name | Employee name |
| Department | Department assigned |
| Job Title | Employee designation |
| Experience | Years of experience |
| Salary | Employee salary |
| Performance Rating | Employee performance score |
| Work Mode | Remote / Hybrid / On-site |
| Employment Status | Active / Resigned / Terminated |
| Joining Date | Employee joining date |
